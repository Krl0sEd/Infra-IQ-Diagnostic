# 📊 Infra-IQ Diagnostic

Interactive tool designed to assess enterprise IT infrastructure, identify security bottlenecks (Firewall, Connectivity, Wi-Fi), and generate automated consulting reports.

## 🚀 Technologies Used
* **Frontend/Backend:** Python with Streamlit
* **Cloud:** Microsoft Azure (App Service)
* **Intelligence:** OpenAI API (for generating personalized insights)
* **Infrastructure:** Docker (for future containerization)

## 🎯 Objective
Automate the IT consulting pre-sales process by transforming a static form into an interactive experience that qualifies leads and delivers immediate value.

## 🛠️ How to run locally
1. Clone the repository
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
